# 🎮 COLOR REFLEX

Jogo viciante baseado no **Efeito Stroop** - seu cérebro vai entrar em conflito!

## 🧠 Como Funciona

Uma palavra de cor aparece na tela, mas a COR DO TEXTO é diferente da palavra escrita.

**Exemplo:**
- Aparece a palavra "AZUL" em cor VERMELHA
- Você deve pressionar **R** (tecla da cor VERMELHA, não da palavra)

## ⌨️ Controles

- **R** = Vermelho
- **B** = Azul
- **V** = Verde
- **A** = Amarelo
- **M** = Magenta
- **C** = Ciano
- **ESC** = Voltar ao menu

## 🎯 Mecânicas Viciantes

- ⏱️ **Tempo limitado** que diminui a cada nível
- 🔥 **Sistema de Streak** - acertos consecutivos dão bonus
- 💯 **Pontuação crescente** com multiplicadores
- 💔 **3 vidas** - erre e perde uma
- 📈 **Níveis progressivos** - fica cada vez mais rápido
- 🏆 **High Score** persistente

## 🚀 Como Jogar

```bash
cd ColorReflex
dotnet run
```

## 💡 Por Que É Viciante?

O **Efeito Stroop** cria um conflito cognitivo natural:
- Seu cérebro quer LER a palavra
- Mas você precisa IDENTIFICAR a cor
- Isso cria tensão mental + satisfação ao acertar
- Respostas rápidas = flow state
- "Só mais uma tentativa" = loop viciante

Divirta-se! 🎮✨
